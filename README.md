# Socket.Client
A Electron WebSocket Client.

## Installation

Use a package manager of your choice (npm, yarn, etc.) in order to install all dependencies

```bash
npm i
```

```bash
yarn
```

## Usage
To run this project 1 scripts will need to be executed `dev`, run it.

```bash
npm run dev
```
```bash
yarn dev
```
